Universal Render Pipeline upgrade
---------------------------------

Supported version: 7.1.8 and up only!

1) Import Suburban House to your URP project, and run SH_URP unitypackage!
2) Enjoy!

High Definition Render Pipeline upgrade
---------------------------------------

Supported version: 7.1.8 and up only!

1) Import Suburban House to your configured HDRP project!
2) Run the auto-upgrade tool on it (Edit/Render pipeline/project materials to HDRP)
3) When done, run SH_HDRP unitypackage to set everything up for HDRP.
4) Enjoy!
